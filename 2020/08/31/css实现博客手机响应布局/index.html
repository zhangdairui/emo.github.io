css实现博客手机响应布局
@emo 2020-08-31 15:38:15 #blog建设记录

今天使用css的【@media screen and (max-width: 980px){}】把手机端的页面也做了适配。
其中，用vw代替px这个单位，会对更多手机机型适配的更好一些。
在适配的过程中，直接用chromeF12自带的响应式布局浏览进行。在字体上不太准确。还好手边有手机可以实际上测试。
（打开hexo server，手机和电脑接入同一wifi，然后把手机浏览器的地址输入：ip地址:4000，就可以在手机上实际调试页面）

遇到的无法解决的问题是ios下safari的滚动条总是默认隐藏的，目前在网上搜到的所有方法没有能够解决这个问题。我还挺想要一个滚动条做装饰的。
